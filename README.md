# AWS Cloud Practitioner – Learning Journey

This repository represents my *AWS Cloud Practitioner* learning journey, focused on building a strong conceptual foundation along with *hands-on AWS console practice*.

The goal of this journey was not only to prepare for the certification, but to truly understand how cloud services work in real-world scenarios.


## Journey Objective:

- Understand core cloud computing concepts
- Learn AWS global infrastructure and services
- Gain hands-on experience using the AWS Management Console
- Strengthen knowledge through practical assignments
- Earn AWS Cloud Practitioner badge and certification
- Build confidence for advanced cloud and DevOps learning paths


## Certification & Recognition:

- AWS Cloud Practitioner Badge (earned)
- AWS Cloud Practitioner Certificate (earned)

These achievements validate both theoretical understanding and practical exposure to AWS fundamentals.


## Topics Covered:

### Cloud Concepts:
- Cloud computing fundamentals
- Benefits of cloud adoption
- AWS Shared Responsibility Model
- AWS global infrastructure (Regions, Availability Zones, Edge Locations)


### Security & Identity:
- AWS IAM fundamentals
- Users, Groups, Roles, and Policies
- Security best practices
- Identity and access control basics


### Core AWS Services:
- Amazon EC2 – compute fundamentals
- Amazon S3 – object storage concepts
- Amazon RDS – database basics
- Amazon VPC – networking fundamentals
- Amazon CloudWatch – monitoring and logging
- AWS Pricing & Billing models


## Hands-On Practice & Assignments

This journey emphasized learning by doing.  
Hands-on tasks and assignments were completed using the AWS Console to reinforce each concept.

### Practical Experience Includes:
- Launching and managing EC2 instances
- Creating and configuring S3 buckets
- Managing IAM users and permissions
- Exploring AWS billing and cost management tools
- Understanding real AWS workflows and service interactions

Assignments were designed to strengthen problem-solving skills and practical understanding rather than rote learning.


## Learning Approach:

- Concept-first, practice-driven learning
- Real AWS account usage
- Console-based exploration
- Focus on clarity over memorization
- Continuous review and self-assessment

This approach helped in developing a clear, long-term understanding of AWS fundamentals.


## Outcome:

- Solid foundation in AWS cloud concepts
- Confidence in navigating AWS services
- Improved understanding of security and pricing
- AWS Cloud Practitioner certified
- Ready for advanced cloud, DevOps, and automation learning


## Next Steps:

- AWS Solutions Architect (Associate)
- DevOps and cloud automation tools
- Cloud security fundamentals
- Hands-on cloud projects
